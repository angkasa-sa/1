# References
This is a list of topics and tools you should be comfortable with before you start.

## Tools To Set Up
Install and verify these on your machine:

- Bun
- Docker
- A code editor you are comfortable with

You should be able to run:

```bash
bun --version
docker --version
docker compose version
```

## Related Topics
Familiarize yourself with:
- NestJS modules, controllers, services, providers, and dependency injection
- TypeORM, including entities, repositories, relations, migrations/synchronization basics, and query
  builders
- REST API design for CRUD resources
- Request validation and response DTOs
- Basic SQL concepts such as joins, filtering, ordering, limits, and uniqueness constraints
- Layered or onion-style architecture: controllers at the edge, business logic in services, data
  access isolated behind repositories/helpers
- React components, hooks, state, effects, and controlled forms
- TanStack Router routing basics
- TanStack React Query mutation basics
- Jotai basics for shared client state
- Feature-based React structure
- Separation of concerns
- SOLID principles at a practical level
- Reusable code vs premature abstraction
- DTOs and API boundaries
- Derived data vs stored data
- Handling loading, error, retry, and empty states
